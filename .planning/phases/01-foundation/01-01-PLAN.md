  <action>
Run `npm create vite@latest . -- --template react-ts` in the project root (the `.` installs into current directory, not a subdirectory).

After scaffold completes, install all dependencies in one command:
```
npm install react-router-dom recharts lucide-react @anthropic-ai/sdk
npm install -D tailwindcss @tailwindcss/vite
```

Create `.env.example` with:
```
VITE_ANTHROPIC_API_KEY=your-api-key-here
```

Update `.gitignore` to include:
- `.env` and `.env.local` (API keys)
- `historical_estimates/` (2GB of XLSX files)
- `*.log` (error logs from Python scripts)
- Keep the standard Vite ignores (node_modules, dist)

Update `vite.config.ts` to include the Tailwind plugin:
```ts
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [react(), tailwindcss()],
})
```

Replace the contents of `src/App.tsx` with a minimal placeholder that imports Tailwind:
```tsx
function App() {
  return <div className="p-8 text-lg">Event Estimate Engine</div>
}
export default App
```

Add Tailwind import to `src/main.tsx` (or the main CSS file):
Add `@import "tailwindcss";` to `src/index.css` (replacing default Vite styles).

<task type="auto">
  <name>Task 2: Initialize shadcn/ui and configure enterprise theme</name>
  <files>components.json, src/lib/utils.ts, src/index.css, tailwind.config.ts (if needed), tsconfig.json, tsconfig.app.json</files>
  <action>
Initialize shadcn/ui:
```
npx shadcn@latest init
```
When prompted, select: New York style, Zinc base color, CSS variables yes. This creates `components.json` and `src/lib/utils.ts`.

NOTE: shadcn/ui init may require path aliases. If it asks to update tsconfig, allow it. The typical setup adds `@/*` alias pointing to `./src/*` in both tsconfig.json and vite.config.ts (via `resolve.alias`).

After init, install a few base components we'll need across all views:
```
npx shadcn@latest add button card table tabs input badge separator
```

Update `src/index.css` CSS variables to establish the enterprise theme. The Zinc base from shadcn is close to what we want. Adjust accent colors:
- Primary: Blue (hsl 221 83% 53%) for actions and links
- Accent: Emerald (hsl 160 84% 39%) for positive indicators/success states
- Keep the Zinc neutrals for backgrounds, borders, text
- Sidebar will use dark background (zinc-900/950) — handled in component CSS, not theme vars

Verify the theme by updating `src/App.tsx` to render a Button component:
```tsx
import { Button } from '@/components/ui/button'

function App() {

      <Button>Test Button</Button>
    </div>
  )
}
export default App
```
